## Notes on using DA with Universal Editor

These are Charity's observations as of March 2026.

- if you delete a block in UE, it might not delete from DA. The block will show but have no content.
- using placeholder text for the UE in DA doesn't seem to work like it does for UE in Xwalk.
- If you have optional rows in your block (such as the 'author' row for the Quote block), you will need to add them by default in UE if you expect to be able to author it. You can't add those ad-hoc, and these rows will just be empty in DA.
